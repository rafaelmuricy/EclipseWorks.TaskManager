## Como rodar o projeto

Apenas aperte play no visual studio. O projeto está configurado com a variavel de ambiente "ConnectionString" em todas as formas, inclusive no container docker e também no Dockerfile.

## Fase 2: Refinamento

Uma melhoria que seria crucial para o projeto seria a possibilidade de upload de imagens para as tarefas, e também para comentários.
Além disso, seria interessante implementar um sistema de notificações, onde o usuário poderia receber notificações por email quando uma tarefa fosse criada ou atualizada.

## Fase 3: Final

A fase 3 do projeto seria uma especificação de como os dados do usuário viriam para a API, e como a API identificaria o nível de permissão que o usuário tem.
